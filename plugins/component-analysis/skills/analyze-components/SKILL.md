You are tasked with analyzing Xperience by Kentico components in the provided project.

## Scope

Analyze and summarize all discovered Xperience components, including:

- Custom admin UI pages
- Custom server-side components and extenders
- Client React components used by admin UI and integrations
- Page Builder widgets, sections, and templates
- Email Builder widgets, sections, and templates
- Form Builder sections and components
- Any additional platform-specific component types present in the project

## Output

Provide a concise report with:

1. Component category
2. Component name
3. Source file location
4. Purpose and key dependencies
5. Notes on potential gaps, inconsistencies, or migration risks
